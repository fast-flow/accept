html5 input accept (20170523)

Notice:
accept="--/*" 会有6秒左右延迟, 采用详细枚举的方式

.** 直接文件名后缀
(chrome firefox 支持所有.**后缀文件名)
(Safari 一个都不支持)

accept=".gif"
accept=".less"
accept=".md"
accept=".jpg" (chrome:*.jpe,*.jpeg,*.jpg)(firefox:*.jpe)
accept=".jpeg" (chrome:*.jpe,*.jpeg,*.jpg)(firefox:*.jpeg)
accept=".jpe" (chrome:*.jpe,*.jpeg,*.jpg)(firefox:*.jpe)
accept=".svg" (chrome 延迟较大)
accept=".php"
accept=".pdf"
accept=".zip"
accept=".7z"
accept=".rar" (chrome 延迟较大)
accept=".html" (chrome:*.html,*.htm)(firefox:*.html)
accept=".htm" (chrome:*.html,*.htm)(firefox:*.htm)

1. image

1.1 accept="image/*" (不建议使用)
1.2 accept="image/gif"
1.3 accept="image/jpeg" (chrome,safari:*.jpe,*.jpeg,*.jpg)(firefox:*.jpg)
1.3 accept="image/jpe" (chrome:*.jpe,*.jpeg,*.jpg)(safari,firefox不支持)
1.3 accept="image/jpg" (chrome:*.jpe,*.jpeg,*.jpg)(safari,firefox:*.jpeg)
1.4 accept="image/jp2"
1.5 accept="image/png"
1.6 accept="image/svg" (chrome,firefox,safari 不支持)
1.7 accept="image/bmp"
1.8 accept="image/gif,image/jpg,image/jpeg,image/png,image/svg"
1.9 accept="image/vnd.dwg" (*.dwg) (chrome,safari,firefox 不支持)
1.10 accept="image/vnd.dxf" (*.dxf) (chrome,safari,firefox 不支持)

2. audio

2.1 accept="audio/*" (不建议使用)
2.2 accept="audio/3gpp" (safari,firefox 不支持)
2.3 accept="audio/ac3"
2.4 accept="audio/basic" (*.au)
2.5 accept="audio/mpeg" (chrome,safari:*.mp2,*.mp3)(firefox:*.mp3)
2.6 accept="audio/mp4" (*.mp4)

3. video

3.1 accept="video/*" (不建议使用)
3.2 accept="video/3gpp" (safari,firefox 不支持)
3.3 accept="video/mpeg" (chrome,safari:*.mpeg,*.mpg)(firefox:*.mpeg)
3.4 accept="video/mp4"

4. text

4.1 accept="text/*" (不建议使用)
4.2 accept="text/css"
4.3 accept="text/csv"
4.4 accept="text/html" (chrome,safari:*.htm,*.html)(firefox:*.html)
4.4 accept="text/htm" (chrome,safari,firefox不支持)
4.5 accept="text/javascript" (*.js) (chrome 延迟较大)
4.6 accept="text/rtf"
4.7 accept="text/plain" (chrome: *.txt,*.csv,*.js,*.php)(safari,firefox:*.txt)
4.8 accept="text/xml" (chrome: *.xml,*.xhtml)(safari,firefox:*.xml) (chrome 延迟较大)

5. application

5.1 accept="application/*" (不建议使用)
5.2 accept="application/msword" (*.doc)
5.3 accept="application/xml-dtd" (*.dtd) (chrome,safari,firefox 不支持)
5.3 accept="application/dtd" (*.dtd) (chrome,safari,firefox 不支持)
5.4 accept="application/javascript" (*.js) (safari,firefox 不支持)(chrome 延迟较大)
5.5 accept="application/json"
5.6 accept="application/vnd.ms-powerpoint" (*.ppt)
5.7 accept="application/rtf" (*.rtf) (chrome,safari 不支持)
5.8 accept="application/xhtml+xml" (*.xhtml)
5.9 accept="application/vnd.ms-excel" (*.xls) (firefox 不支持)
5.10 accept="application/xml" (chrome:*.xml,*.xhtml)(safari,firefox:*.xml) (chrome 延迟较大)
5.11 accept="application/zip" (chrome 延迟较大)
